CESAR

The Cesar web application provides a low-level way to do syntactic research in annotated text corpora.

Learn more »

What can I do?

The Cesar application offers a set of tools that allow the researcher to work with syntactically annotated texts:

  • Define and host corpora. See the currently hosted list in Browse/CorpusParts.
  • Look for texts satisfying metadata criteria such as genre and language. See Browse/AllTexts.
  • Review the syntactic annotation of a sentence in a text (see the example at the bottom right).
  • Define a search that takes into account syntactic conditions. See Search/Specify or Search/List
  • Specify how to have output features calculated for search hits
  • Make searches available for other Cesar users
  • Execute a search on a part or the whole of any of the corpora available
  • Download results as CSV, Excel or SQlite database
  • Review the results online as KWIC, review the output for each hit. See Search/Results

Getting started

The CESAR application has been designed by the Technical Service Group of the Humanities Lab at the Radboud University Nijmegen. Any researcher is invited to register and maintain his or her own research projects. Projects of one researcher are by default not visible to other researchers, but it is possible to specify a different accessibility level.
