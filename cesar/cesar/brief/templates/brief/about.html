Background

The Project Brief app has been developed by SIL-Bolshoi

Advanced help

There are basically two modes of operation for this app.

  • Convenient: keep the project brief data on the server. Downloading can be done any time, but would only serve backup purposes.
  • Secure: upload and download the project brief data when editing or viewing is needed. Make sure to remove the data from the server, once it has been successfully downloaded.

Technical details

This implementation of the project 'brief' is a Django app under the larger 'Cesar' web application. The source code of Cesar is available at github. The 'brief' source code can be found in there. The 'brief' also makes use of the 'basic' app (providing the standardized list view and details view)
